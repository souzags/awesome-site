O algoritmo

O Crivo de Eratóstenes funciona da seguinte maneira:

  1. Crie uma lista de inteiros consecutivos de 2 até n;
  2. Seja p = 2, o menor número primo;
  3. Marque todos o múltiplos de p (2p, 3p, 4p...) na lista (o p não deve ser marcado);
  4. Encontre o menor inteiro maior que p que não esteja marcado. Se não houver, pare. Caso contrário, faça p receber esse número e repita a partir do passo 3;

Quando o programa termina, os números que não estão marcados são primos.

O código


    function getPrimes() {
        maxNumber = parseInt(document.getElementById("calcInput").value);
        output = document.getElementById("output-container");
        n = new Array(maxNumber + 1);
    
        n.fill(true);
        output.innerHTML = "";
    
        n[0] = n[1] = false;
    
        for (let i = 2; i < n.length - 1; i++) {
    
            if (n[i]) {
                for (let j = i + 1; j < n.length; j++) {
                    if (j % i == 0) {
                        n[j] = false;
                    }
                }
            }
        }
    
        for (let i = 0; i < n.length; i++) {
            if (n[i]) {
                output.innerHTML += "<p>" + i + "</p>";
            }
        }
    }
